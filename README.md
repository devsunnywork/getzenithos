## ğŸš€ Quick Start Guide

### Prerequisites
- Node.js (v16.0+)
- MongoDB Atlas Account
- Git

### Installation Sequence

1. **Protocol Initialization**
   ```bash
   git clone https://github.com/devsunnywork/zenith-os.git
   cd zenith-os
   ```

2. **Dependency Synchronization**
   ```bash
   cd server
   npm install
   ```

3. **Core Configuration**
   Create a `.env` file in the `server/` root:
   ```env
   MONGO_URI=your_mongodb_atlas_connection_string
   JWT_SECRET=your_ultra_secure_secret_key
   PORT=5000
   ```

4. **System Launch**
   ```bash
   node server.js
   ```

5. **Access Terminal**
   - Head to `http://localhost:5000`
   - Default admin protocols will initialize on the first boot.
